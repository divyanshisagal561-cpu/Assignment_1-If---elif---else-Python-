# Python `if-elif-else` Practice Questions 🐍

A beginner-friendly collection of **31 Python practice programs** focused on mastering the `if`, `if-else`, `if-elif-else`, nested `if`, and logical operators. These exercises are designed to strengthen decision-making and problem-solving skills in Python.

---

## About the Project

This repository contains solutions to 31 practice questions that cover the fundamentals of conditional statements in Python.

The goal of this project is to build a strong understanding of:

- `if` statements
- `if-else`
- `if-elif-else`
- Nested `if`
- Logical operators (`and`, `or`, `not`)
- Comparison operators
- Real-world decision-making problems

This project is ideal for **Python beginners** who want to practice writing conditional logic.
